# What Is in Git vs What You Download Locally

This document answers the common supervisor question: *“Is the real data pushed to GitHub?”*

**Short answer: No — and that is intentional and correct for an MSc capstone.**

---

## What IS pushed to GitHub (your supervisor can clone and review)

| Item | Location | Purpose |
| :--- | :--- | :--- |
| All source code | `*.py`, `experiments/` | Reproducible implementation |
| Documentation | `docs/`, `README.md` | Thesis tables, limitations, examiner checklist |
| Dataset guides | `data/*/PLACE_DATA_HERE.txt` | Official URLs + expected filenames |
| Download script | `download_data.py` | One-command fetch from public mirrors |
| **Verified result reports** | `results/*.json` | Metrics from real-data runs (committed for review) |
| **Thesis figures** | `results/figures/` | PNG + PDF plots for thesis chapter |
| Unit tests | `tests/` | Automated sanity checks |

---

## What is NOT pushed to GitHub (by design)

| Item | Why excluded | How to obtain |
| :--- | :--- | :--- |
| NASA `.mat` files (~56 MB) | Large binary; public NASA dataset | `python download_data.py --nasa` |
| Oxford `.mat` (~266 MB) | Large binary; Oxford ORA repository | `python download_data.py --oxford` |
| CALCE `.xlsx` logs (~50 MB) | Large binary; CALCE UMD archives | `python download_data.py --calce` |
| Model checkpoints (~3 MB) | Regenerated on training; not needed for code review | `python run_experiments.py` |
| `validation_predictions.json` | Large intermediate file for figure regeneration | Generated by `generate_figures.py` |

These paths are listed in `.gitignore`. This follows standard academic practice: **publish code + results + reproduction instructions**, not multi-hundred-megabyte raw datasets in git.

## How the supervisor verifies “real data” was used

1. Open `results/experiment_comparison_report.json` → field `data_sources` lists `real_nasa_mat`, `real_oxford_mat`, `real_calce_xlsx` with cycle counts.
2. Run `python scripts/verify_setup.py` on any machine — reports which datasets are present locally.
3. Re-run `python run_experiments.py` after `download_data.py --all` — NASA SOH RMSE should be ~**0.022** (±0.002).
